This is the model code associated with the paper:

Gaines JL, Finn KE, Slopsema JP, Heyboer LA, Polasek KH (2018) A model
of motor and sensory axon activation in the median nerve using surface
electrical stimulation. J Comput Neurosci

These NEURON files were contributed by KH Polasek.  This model
requires data files to run which are not currently included in the
archive.
